# Security Policy

## Supported Versions

Security updates are provided for the latest stable version of the plugin.

Older versions are not actively supported. Users are strongly encouraged to update to the latest available version.

| Version | Supported |
| ------- | --------- |
| Latest stable version | ✅ |
| All previous versions | ❌ |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability, please report it privately by contacting us at:

support@wpovernight.com

Please include as much detail as possible, including:

- A clear description of the issue
- Steps to reproduce it
- The affected plugin version
- Any relevant screenshots, logs, or proof-of-concept details

We will review the report and follow up as soon as possible.

## Disclosure

Please allow us reasonable time to investigate and release a fix before publicly disclosing the vulnerability.
